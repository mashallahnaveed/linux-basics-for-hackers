## 🧪 Practical Examples

```bash
cat file.txt | grep "error"
```

👉 Filters lines containing "error"

```bash
tail -n 10 log.txt
```

👉 Shows last 10 lines of logs

```bash
ps aux | grep firefox
```

👉 Finds running Firefox process
